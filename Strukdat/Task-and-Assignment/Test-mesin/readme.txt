(LTS01) List Tunggal Statis


Pembuat Soal: Tim Asistem Pemograman 6

Batas Waktu Eksekusi	5 Detik
Batas Memori	1 MB

Buatlah sebuah list tunggal statis untuk menyimpan nama - nama karyawan dengan atribut nama, umur dan jenis kelamin.
Lakukan perintah - perintah list dengan urutan berikut :

addFirst
addFirst
addLast
delLast
print
addAfter L.data[L.data[L.first].next].next
print
addFirst
deleteAfter L.first
delLast
print
delAll
print

Format Masukan :
nama, umur dan jenis kelamin sebanyak 5

Format Keluaran :
data karyawan dan disetiap command print ditambah pembatas berupa ====== di baris akhir print

Contoh Masukan

Fikry 20 L
Faisal 30 L
Mira 10 P
Ridwan 17 L
Fauzi 21 L


Contoh Keluaran

Faisal 30 L
Fikry 20 L
======
Ridwan 17 L
Faisal 30 L
Fikry 20 L
======
Fauzi 21 L
Faisal 30 L
======
List Kosong!
======
