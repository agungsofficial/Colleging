Kemampuan yang dievaluasi: kemampuan memahami konsep mesin karakter.
Hitunglah urutan huruf pada pita yang terdiri dari konsonan-vokal- angka.
Format Masukan:

String pita 1000 char.
Format Keluaran:

integer jumlah urutan huruf pada pita sesuai soal.
Contoh Masukan

ak7 adala6 anak ge3bala selalu r1ang se9ta gembira.


Contoh Keluaran

3
