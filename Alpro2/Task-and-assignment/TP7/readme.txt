Hitung hitung IP ceria

Piyu adalah seorang mahasiswi Ilmu Komputer yang sangat rajin. Menjelang akhir smester genap ini, teman-teman piyu sedikit kebingungan untuk menghitung Indeks Prestasi yang akan mereka dapatkan di smester ini, mendengar hal itu Piyu pun berencana untuk membuat sebuah program penghitung Indeks Prestasi (IP) untuk teman-temannya. Hal yang pertama Piyu lakukan adalah membaca 2 buah list record (firstlist.txt dan secondlist.txt) yang berisi NIM, nama, jumlah mutu, dan jumlah sks yang teman-temannya kontrak.
Contoh record: 1403407 Lyra 71,60 20
*1403407 = NIM
*Piyu = Nama
*71,60 = Jumlah mutu
*20 = Jumlah sks yang dikontrak
Setelah Piyu mendapatkan semua data dari kedua list tersebut, Piyu harus membuat sebuah list (result.txt) dengan format NIM, Nama, total IP dan diurutkan berdasarkan IP terbesar, bila IP nya sama maka diurutkan berdasarkan NIM secara descending.

Contoh: 1403407 Lyra 3,58
*1403407 = NIM
*Piyu = Nama
*3,58 = Total IP = (jumlah mutu / jumlah sks)
Bantulah Piyu untuk membuat program tersebut.
